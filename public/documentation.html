Documentation Page

To build the server-side component of a “movies” web application. The web application will provide users with access to information about different movies, directors, and genres. Users will be able to sign up, update their personal information, and create a list of their favorite movies.


Example: "Your account was successfully deleted."
Request URL HTTP Method Request body data format Response body data format
Return a list of ALL movies to the user /movies GET None A JSON array holding data about all movies
Return data (description, genre, director, image URL, whether it’s featured or not) about a single movie by title to the user /movies/:title GET None A JSON object holding data about a single movie, containing the title, genre, director and release year.
Example: { title: 'Harry Potter and the Sorcerer's Stone', genre: ['Adventure', 'Family', 'Fantasy'], director: 'Chris Colombus', releaseYear: 2001 }
Return data about a genre (description) by name/title (e.g., “Thriller”) /movies/:title/:genre GET None A text message indicating the genre(s) of a single movie
Example: "Harry Potter Series are Adventure, Family and Fantasy movies."
Return data about a director (bio, birth year, death year) by name /movies/directors/:name GET None A text message indicating data about a director
Example: "Chris Colombus was born on September 10, 1958 in Spangler, Pennsylvania, USA. He is a director, producer, and screenwriter, known for Home Alone (1990), Harry Potter and the Sorcerer's Stone (2001) and its sequel, Harry Potter and the Chamber of Secrets (2002)."
Get all users /users GET None A text message indicating data about users
Get a user by username /users/:Username GET None A text message indicating data about finding a user
Allow new users to register /users POST A JSON object holding data about the new user
Example: { Username: '2pac', Password: 'westside123', Email: 'tashakur@gmail.com', Birthday: "15/06/1971" } 		A JSON object holding data about the new user
Example: { id: 313abc8798298eetwewe222445, firstName:'Tupac', lastName: 'Shakur' username: '2pac', password: 'westside123', emailAddress: 'tashakur@gmail.com' }
Update a user's info, by username /users/:Username PUT None A text message indicating data about updating a user by username
Add a movie to a user's list of favorites /users/:Username/movies/:MovieID POST None A text message indicating data about adding a movie by movieID to the favorites of users
Remove movie from the favorite list with MovieID. /users/:Username/favorites/:MovieID DELETE None A text message indicating movie was successfully remove.
Delete a user by username /users/:Username DELETE None A text message indicating the user is successfully removed by username
